# 🕒 Hora do Dia

Projeto simples feito com HTML, CSS e JavaScript.

Ele mostra:
- Hora atual
- Minutos exatos
- Imagem dinâmica de acordo com o horário (manhã, tarde ou noite)

## 🚀 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📸 Funcionalidades

O sistema altera:
- Texto
- Imagem
- Estilo visual

Baseado na hora do sistema.
